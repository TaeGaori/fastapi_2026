# 로또 1등 자동 당첨 판매점 데이터 수집 및 PostgreSQL 적재 프로그램

공공데이터포털의 '로또 1등 자동 당첨 판매점 현황' API를 활용하여 데이터를 페이지별로 자동 수집하고, 데이터 가공 및 유효성 검증을 거쳐 PostgreSQL 데이터베이스에 안전하게 저장하는 파이썬 프로그램입니다.


## 1. 주요 기능
- 특정 상호의 자동 복권 1등 당첨 횟수를 알 수 있다.
- 원하는 지역의 복권매장 상호명과 당첨횟수를 알 수 있다.(서울)


## 1. 파라미터 매핑 가이드 (로또 API 전용)
공공데이터 API에서 자주 쓰이는 파리미터명과 다르게 작동
* `pageNo` -> **`page`**
* `numOfRows`-> **`perPage`**
* `_type` -> **`returnType`**



## 2. 컬럼 가공 및 삭제
- '지역' 컬럼의 앞 2글자만 잘라내어 새 컬럼 생성 (예: '서울 서초구' -> '서울')
        df_2['시도'] = df_raw['지역'].str[:2]

- 불필요한 '순번' 컬럼 가차 없이 삭제
        if '순번' in df_raw.columns:
            df_raw.drop('순번', axis=1, inplace=True)
